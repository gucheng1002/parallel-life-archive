# 平行人生档案

手机端优先的 Next.js + Tailwind CSS 可点击原型。第一版只使用本地假数据，不接 AI API、不接数据库、不做登录。

## 运行

```bash
npm install
npm run dev
```

默认预览地址：

```bash
http://127.0.0.1:3000
```

## 已实现流程

- intro 开场动画
- gateway 四节点选择
- transition 时间线偏移转场
- start 人生线开始页
- quiz 6 道单选 + 短输入问答
- loading 生成等待页
- result 假数据结果页
- share 分享截图页
